SKILLFUL LIBRARY SETUP
======================

App Name:
Skillful

Package Name:
com.skillful.progress

1. Copy the included lib folder into your Flutter project.

2. Add this dependency to pubspec.yaml:

dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.5.3

3. Run:

flutter clean
flutter pub get
flutter run

Included:
- Splash screen
- Dashboard
- Skills management
- Skill details
- Practice sessions
- Goals
- Statistics
- Achievements
- Settings
- Light and Dark Mode
- SharedPreferences local storage
- Privacy Policy
- Terms & Conditions

Optional logo asset:
assets/logo.png
